#### (1) 라이브니스 프로브 생성 - 정상
> mynapp-pod-liveness.yml
```yaml
apiVersion: v1
kind: Pod
metadata:
  name: mynapp-pod-liveness
spec:
  containers:
  - image: c1t1d0s7/myweb
    name: mynapp
    ports:
    - containerPort: 8080
      protocol: TCP
    livenessProbe:
      httpGet:
        path: /
        port: 8080
```

```
kubectl create -f mynapp-pod-liveness.yml
```

```
kubectl get pods --watch
```

#### (2) 라이브니스 프로브 생성 - 비정상
