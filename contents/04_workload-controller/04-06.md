## 6. 크론잡

### 1) 크론잡 소개
크론잡(CronJob)

### 2) 크론잡 생성
> mynapp-cjob.yml
```yaml
apiVersion: batch/v1beta1
kind: CronJob
metadata:
  name: mynapp-cjob
spec:
  schedule: "*/2 * * * *"
  jobTemplate:
    spec:
      template:
        metadata:
           labels:
             app: mynapp-cjob
        spec:
          restartPolicy: OnFailure
          containers:
          - name: mynapp
            image: busybox
            command: ["sleep", "60"]
```

```
kubectl create -f mynapp-cjob.yml
```

### 3) 크론잡 확인
```
kubectl get cronjobs.batch
```

kubectl get pods
