## 5. 잡

### 1) 잡 소개
잡(Job) 컨트롤러는

### 2) 잡 생성
> mynapp-job.yml
```yaml
apiVersion: batch/v1
kind: Job
metadata:
  name: mynapp-job
spec:
  template:
    metadata:
      labels:
        app: mynapp-job
    spec:
      restartPolicy: OnFailure
      containers:
      - name: mynapp
        image: busybox
        command: ["sleep", "60"]
```

```
kubectl create -f mynapp-job.yml
```

### 3) 잡 확인
```
kubectl get jobs.batch
```

```
kubectl get pods
```

```
kubectl get pods
```

```
kubectl get jobs.batch
```
