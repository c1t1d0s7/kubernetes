## 3. hostPath

### 1) hostPath 볼륨 소개

### 2) hostPath 볼륨 생성
```yaml
volumes:
- name: <VOL-NAME>
  hostPath:
    type: <TYPE>
    path: <PATH>
```

### 4) hostPath 볼륨을 사용하는 레플리카셋 생성
> mynapp-rs-hostpath.yml
```yaml
apiVersion: apps/v1
kind: ReplicaSet
metadata:
  name: mynapp-rs-hp
spec:
  replicas: 2
  selector:
    matchLabels:
      app: mynapp-rs-hp
  template:
    metadata:
      labels:
        app: mynapp-rs-hp
    spec:
      containers:
      - name: web-server
        image: nginx:alpine
        volumeMounts:
        - name: web-content
          mountPath: /usr/share/nginx/html
        ports:
        - containerPort: 80
      volumes:
      - name: web-content
        hostPath:
          type: Directory
          path: /web_contents
```

> mynapp-svc-hostpath.yml
```yaml
apiVersion: v1
kind: Service
metadata:
  name: mynapp-svc-hp
spec:
  ports:
  - port: 80
    targetPort: 80
  selector:
    app: mynapp-rs-hp
```

```
kubectl create -f mynapp-rs-hostpath.yml -f mynapp-svc-hostpath.yml
```
