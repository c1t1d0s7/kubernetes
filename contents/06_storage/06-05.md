### 2) PV 생성 및 확인
> mynapp-pv-nfs.yml
```yaml
apiVersion: v1
kind: PersistentVolume
metadata:
  name: mynapp-pv-nfs
spec:
  capacity:
    storage: 5Gi
  accessModes:
    - ReadWriteMany
  persistentVolumeReclaimPolicy: Retain
  nfs:
    path: /nfs-volume #Modify
    server: X.X.X.X #Modify
```

```
kubectl create -f mynapp-pv-nfs.yml
```

```
kubectl get persistentvolumes
```

### 3) PVC 생성 및 확인
> mynapp-pvc-nfs.yml
```yaml
apiVersion: v1
kind: PersistentVolumeClaim
metadata:
  name: mynapp-pvc-nfs
spec:
  accessModes:
    - ReadWriteMany
  resources:
    requests:
      storage: 5Gi
  volumeName: mynapp-pv-nfs
```

```
kubectl create -f mynapp-pvc-nfs.yml
```

```
kubectl get persistentvolumeclaims
```

```
kubectl get persistentvolumes
```

### 4) 레플리카셋 컨트롤러 생성 및 확인
> mynapp-rs-nfs.yml
```yaml
apiVersion: apps/v1
kind: ReplicaSet
metadata:
  name: mynapp-rs-nfs
spec:
  replicas: 2
  selector:
    matchLabels:
      app: mynapp-rs-nfs
  template:
    metadata:
      labels:
        app: mynapp-rs-nfs
    spec:
      containers:
      - name: mynapp
        image: nginx
        volumeMounts:
        - name: nfs-share
          mountPath: /usr/share/nginx/html
        ports:
        - containerPort: 80
      volumes:
      - name: nfs-share
        persistentVolumeClaim:
          claimName: mynapp-pvc-nfs
```

```
kubectl create -f mynapp-rs-nfs.yml
```

```
kubectl get pods
```
