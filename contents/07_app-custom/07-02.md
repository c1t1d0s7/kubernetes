### 2) 쿠버네티스에서 인자 변경
> mynapp-pod-arg.yml
```yaml
apiVersion: v1
kind: Pod
metadata:
  name: mynapp-pod-arg
spec:
  containers:
  - name: mynapp
    image: c1t1d0s7/myweb
    args:
    - "Customized Hello World!"
    ports:
    - containerPort: 8080
      protocol: TCP
```

```
kubectl create -f mynapp-pod-arg.yml
```

```
kubectl port-forward mynapp-pod-arg 8080:8080
```

```
curl http://localhost:8080
```
