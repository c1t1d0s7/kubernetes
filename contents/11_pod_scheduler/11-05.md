## 5. 커든

커든(Cordon): 지정된 노드에 추가로 파드를 스케줄링하지 않음

kubectl cordon kube-node1
kubectl cordon kube-node2

$ kubectl get no
NAME           STATUS                     ROLES    AGE   VERSION
kube-master1   Ready                      master   4d    v1.16.8
kube-node1     Ready,SchedulingDisabled   <none>   4d    v1.16.8
kube-node2     Ready,SchedulingDisabled   <none>   4d    v1.16.8
kube-node3     Ready                      <none>   4d    v1.16.8

> mynapp-rs-cordon.yml
```yaml
apiVersion: apps/v1
kind: ReplicaSet
metadata:
  name: mynapp-rs-cordon
spec:
  replicas: 3
  selector:
    matchLabels:
      app: mynapp-rs-cordon
  template:
    metadata:
      labels:
        app: mynapp-rs-cordon
    spec:
      containers:
      - name: mynapp
        image: c1t1d0s7/myweb
```

kubectl get po -o wide
kubectl scale rs mynapp-rs-cordon --replicas=5
kubectl get po -o wide

kubectl uncordon kube-node1
kubectl uncordon kube-node2
