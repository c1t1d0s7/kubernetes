## 3. 클러스터 외부 서비스

```
!!!수정!!!
NAME TYPE CLUSTER-IP EXTERNAL-IP PORT(S) AGE
cccr ClusterIP 10.110.94.122 <none> 80/TCP 6s
kubernetes ClusterIP 10.96.0.1 <none> 443/TCP 24d
!!!수정!!!
```

### 1) 외부 서비스용 레플리카셋 생성 및 확인
없으면 생성!!!
```
kubectl create -f mynapp-rs.yml
```

```
kubectl get replicasets.apps
```

```
kubectl get pods
```

### 2) NodePort 서비스 생성
> mynapp-svc-ext-nodeport.yml
```yaml
apiVersion: v1
kind: Service
metadata:
  name: mynapp-svc-ext-np
spec:
  type: NodePort
  ports:
  - port: 80
    targetPort: 8080
    nodePort: 31111
  selector:
    app: mynapp-rs
```

```
kubectl create -f mynapp-svc-ext-nodeport.yml
```

### 3) NodePort 서비스 확인
```
kubectl get services
```

```
kubectl get endpoints mynapp-svc-ext-np
```

```
curl http://x.x.x.x:31111
```

### 4) LoadBalancer 서비스 생성
> mynapp-svc-ext-loadbalancer.yml
```yaml
apiVersion: v1
kind: Service
metadata:
  name: mynapp-svc-ext-lb
spec:
  type: LoadBalancer
  ports:
  - port: 80
    targetPort: 8080
  selector:
    app: mynapp-rs
```

```
kubectl create -f mynapp-svc-ext-loadbalancer.yml
```

### 5) LoadBalancer 서비스 확인
```
kubectl get services
```

```
curl http://x.x.x.x
```
