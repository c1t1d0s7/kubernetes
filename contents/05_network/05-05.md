## 5. 레디니스 프로브

### 1) 레디니스 프로브 소개
레디니스 프로브(Rediness Probe)

### 2) 레디니스 프로브 생성
> mynapp-rs-readiness.yml
```yaml
apiVersion: apps/v1
kind: ReplicaSet
metadata:
  name: mynapp-rs-readiness
spec:
  replicas: 3
  selector:
    matchLabels:
      app: mynapp-rs-readiness
  template:
    metadata:
      labels:
        app: mynapp-rs-readiness
    spec:
      containers:
      - name: mynapp
        image: c1t1d0s7/myweb
        readinessProbe:
          exec:
            command:
              - ls
              - /var/ready
        ports:
        - containerPort: 8080
```

!!!추가!!!
> mynapp-svc-readiness.yml
```yaml
apiVersion: v1
kind: Service
metadata:
  name: mynapp-svc-readiness
spec:
  ports:
  - port: 80
    targetPort: 8080
  selector:
    app: mynapp-rs-readiness
```

```
kubectl create -f mynapp-rs-readiness.yml -f mynapp-svc-readiness.yml
```
